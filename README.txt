JOHN CONWAY'S GAME OF LIFE AND OTHER CELLULAR AUTOMATA

this program simulates John Conways game of life, as well as cellular automata rules 30 and 82.

Controls:
"Esc": to exit
"Space": to start and stop simulation (simulation starts in "off" state)
"Left Click": click or hold down to draw "alive" cells
"Right Click": click or hold down to kill "alive" cells
"s": cycle through John Conway's Game of Life -> Rule 30 -> Rule 82

Requirements:

Pygame: pip install pygame
Numpy: pip install numpy
